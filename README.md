# Заказы интернет-магазина

#### Проект, решая прикладную задачу на обработку данных, показывает работу с файлами формата JSON: 
#### как считать, записать и обработать информацию для такого типа данных.

### Формулировка задачи, описание входных данных

В файле translator.json в корневом каталоге дана информация о заказах из интернет-магазина за последний месяц:

```
{
     'Номер заказа' : 
     {
         'date' : 'Дата заказа',
         'user_id' : id клиента,
         'quantity' : количество товаров в заказе,
         'price' : стоимость заказа
     },
     
     'Номер заказа' : 
     {
         'date' : 'Дата заказа',
         'user_id' : id клиента,
         'quantity' : количество товаров в заказе,
         'price' : стоимость заказа
     }
     
     ...
 }
```

Считывая и обрабатывая данные из файла, необходимо найти следующую информацию:

1. Какой номер самого дорого заказа за июль?
2. Какой номер заказа с самым большим количеством товаров?
3. В какой день в июле было сделано больше всего заказов?
4. Какой пользователь сделал самое большое количество заказов за июль?
5. У какого пользователя самая большая суммарная стоимость заказов за июль?
6. Какая средняя стоимость заказа была в июле?
7. Какая средняя стоимость товаров в июле?

### Решение задачи

Чтобы обработать данные и ответить на вопросы задачи, первым делом нужно считать данные из файла и преобразовать их в словарь. 

Сделать это можно с помощью методов json.load или json.loads:

```python
import json

with open("translator.json", "r") as my_file:
    orders = json.load(my_file)
```

Для обработки данных будет удобно использовать цикл ```for```, который поможет пройтись по элементам словаря (важно учесть, что информация о заказах хранится во вложеном словаре).

Таким образом с помощью циклов можно работать с данными о заказах и получать нужную информацию.
